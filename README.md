anonymize_cvr.py - Software to anonymize CVRs

Automatically anonymize a CVR (Cast Vote Record) file to meet the Colorado requirement
that styles with fewer than 10 ballots per style must be anonymized to protect voter privacy.

See also [Preserving Anonymity of Cast Vote Record](https://www.sos.state.co.us/pubs/rule_making/written_comments/2018/20180309BranscombEtAl.pdf) -
2018-03-09. Endorsed by John McCarthy, Neal McBurnett, Harvie Branscomb, Ron Rivest, Philip Stark
Colorado Risk-Limiting Tabulation Audit rulemaking.

Note - this is alpha-quality software at best....
